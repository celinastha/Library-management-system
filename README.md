# Library-management-system

## 1. Clone and install
Clone the repository:
- git clone https://github.com/celinastha/Library-management-system <br>
- cd "project-root" -> "Library-management-system" in this case 
- Install dependencies in 2 different terminals
  - cd backend and npm install
  - cd frontend and npm install

  ## 2. Compile and run
Backend (Node.js/Express)
- cd backend
- npm install (if not already)
- node server.js
- Starts the server on PORT (from .env 3000)
Frontend (Vite React App)
- cd frontend
- npm install (if not already)
- npm run dev
- Runs local host (5713 default)
